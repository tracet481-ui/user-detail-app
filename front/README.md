# Vue User Detail App

Bu proje Vue 3 kullanılarak geliştirilmiş bir kullanıcı listeleme ve detay görüntüleme uygulamasıdır.

## Özellikler

- Kullanıcı listeleme
- Kullanıcı detay sayfası
- Posts, Albums ve Todos görüntüleme
- Vue Router ile sayfa geçişleri
- Pinia ile state management
- LocalStorage ile aktif tab bilgisini saklama
- Tailwind CSS ile modern arayüz

## Kullanılan Teknolojiler

- Vue 3
- Vite
- Vue Router
- Pinia
- Tailwind CSS
- JSONPlaceholder API

npm install
